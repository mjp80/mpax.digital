About this site

As your career progresses, you tend to drop out of the real work and migrate towards higher level, more managerial activities.

That has certainly happened in my case, so I wanted to revisit coding by creating the lowest common denominator MVP of a website and then undertake a journey of modernising it and documenting the journey.

Latest updates

  • 15/07/2020 - CSS split

    Simple update today. CSS is now loaded from a separate stylesheet. Best practice says I should probably inline it as well, but that's for another day. Lighthouse still at 100, thankfully!

    Next is to add some more design elements to make it more appealing.

  • 14/07/2020 - Build pipeline

    The site now has the barebones of a modern dev & deployment pipeline set up. I used npm to do it. We have a static HTTP server now serving the dev site through http-server, and code is automatically minified via html-minifier and deployed via Netlify's excellent CLI/API.

    I'll probably split the CSS next, as it's becoming a pain to update both pages separately and inline it. Hopefully my 100% Lighthouse scores won't fall.

  • 13/07/2020 - Site launch

    First iteration is a couple of hardcoded HTML pages with inlined CSS and no JS. Minified manually using an online tool and copied up to Netlify.

    It should score 100 across the board on Lighthouse, be (almost) WCAG-AAA accessible and pass a penetration test with a top grade.

    It's already becoming evident after 2 pages that I need to put together a simple build and deployment pipeline and look at a lightweight CMS to manage content. So that'll be the next job.
